Description

For the first time in the history of Jignasa, Sanketha proudly presents padmavyuha, the line follower robot event.
Padmavyuha is the historical segment. In the event the line follower robot has to follow the path paved by the event coordinators in less time.

Judgement Criteria

The one who has mastery of all rounds is destined as the winner.


Rules

1. Team can have up to two members.
2. Only one participant should control the robot no player can be exchanged in the middle of match.
3. Human interference (e.g. touching the robot) during the game is not allowed.
4. The coordinator can change the rules of the game depending on the situation.
5.On spot registrations is also available.


Team size
3 (Maximum)
